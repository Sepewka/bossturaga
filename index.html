📖 Как пользоваться калькулятором

1. Добавление босса: Выберите босса из списка, затем режим. Количество допок выбирается автоматически из доступных вариантов. Если режим не даёт допок, кнопка добавления будет заблокирована.

2. Таблица только для просмотра: Все изменения вносятся через форму добавления вверху. В таблице нельзя редактировать данные напрямую.

3. Автосохранение: Все данные сохраняются в браузере (localStorage) и восстанавливаются при повторном открытии страницы.

4. Экспорт: Выберите формат (TSV или Excel) и нажмите «Экспорт» — файл скачается автоматически.

5. Импорт: Нажмите «Импорт» и выберите ранее сохранённый файл (TSV или Excel .xls) — таблица восстановится.

6. Очистка: Кнопка «Очистить всё» удаляет всех боссов и очищает локальное хранилище. Также можно удалить выбранных боссов через чекбоксы и кнопку «Удалить выбранных».

ℹ️ Все числа отображаются с разделителями тысяч для удобства чтения.
